Publipostage email simple, local et privé.

Fonctionnalités

  • Éditeur HTML : gras, italique, souligné, listes et liens validés.
  • Modèles réutilisables : sauvegardés localement sur votre machine.
  • Import Excel : les colonnes alimentent automatiquement les champs du message.
  • SMTP de votre choix : configuration locale, mot de passe stocké dans le trousseau système.
  • Envoi maîtrisé : progression en direct, annulation à tout moment, confirmation avant envoi.
  • Mises à jour signées : distribuées directement via GitHub Releases.

Plateformes

  • macOS
  • Windows
  • Linux
